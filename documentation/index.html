Metronic comes with a huge collection of plugins and UI components and it works in all major web browsers, tablets and phones.

Disable Fixed Header

Metronic by default uses fixed header mode. To disable fixed header:

  1. Remove fixed-top class from body element <body class="fixed-top">
  2. Remove navbar-fixed class from header container element <div class="header navbar navbar-inverse navbar-fixed-top">

Default Hidden Sidebar

To set default hidden sidebar just add sidebar-closed class to the page container as shown below: 

<!-- BEGIN CONTAINER -->
<div class="page-container row-fluid sidebar-closed">
  <div class="page-sidebar nav-collapse collapse">
  <!-- SIDEBAR MENU GOES HERE -->
  </div>
</div>
<!-- END CONTAINER -->

9. Javascript Initialization

To top

All javascript initialization implemented in assets/js/app.js thorugh App object as shown below. This approach enables an easy modular implementation to initialize jquery plugins and other application logics. 

var App = function () {

    var localVariable; //local variable

    var sampleFunction = function() {
       //sample function code
    }

    return {

        //main method to initiate template pages
        init: function () {           
            sampleFunction() // call local function
        },

        //sample method declaration
        sampleMethod: function (test) 
        {
          alert(test);
      }

    };

}();
